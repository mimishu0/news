Guangzhou Metro Line 11 is the first circular line in the Guangzhou Metro system. It passes through five districts: Yuexiu, Tianhe, Haizhu, Liwan, and Baiyun, covering key areas in Guangzhou's central urban region. It is 44.2 kilometers long and consists of 31 stations, including 26 interchange stations. The only depot, Chisha Depot, features a two-level design and connects to the network at Chisha and Longtan stations. The rolling stock is 8-car Type A trains, with a maximum speed of 80 km/h and supports fully automated operation.
