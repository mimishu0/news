台湾总统赖清德接受电视媒体专访时说，中国大陆要“并吞”台湾其实不是为了领土的完整，真正目的是“想要改变以规则为基础的世界秩序”以成就霸权。受访学者分析，赖清德希望借此争取国际民主阵营共同抵抗北京；而大陆接下来则会进一步推进台湾问题“内国化”。
他首先再次重申“中华民国跟中华人民共和国本来就互不隶属”论点，并强调“中华民国已经在台澎金马落地生根，民主自由也在台湾成长茁壮，更重要的是民主台湾已经在全世界开枝散叶”。
他说，台湾人民所希望拥有的民主自由生活方式，大陆不能够视为是对其挑战。赖清德并认为，北京并不是因为领土关系才想“侵略台湾”，真正目的是“想要改变以规则为基础的世界秩序”，想在国际上成就霸权。
